# Practicas_procesos

En este proyecto puedes encontrar distintas simulaciones de Paseos Aleatorios Simples (PAS) en las que estudiamos el comportamiento de las trayectorias dependiendo de cual sea la probabilidad (p=0.5, p=0.4, p=0.6).
A continuación, podemos visualizar 20 trayectorias de un PAS, junto con la media teórica y un entorno alrededor de la media de longitud $\pm$1 y $\pm$2 desviaciones típicas teóricas para las probabilidades anteriores.
Por último, visualizamos la media teórica junto con la media empírica cuando promediamos varias trayectorias
